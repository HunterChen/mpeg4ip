What's new in Darwin Streaming Server

    • Accessibility. You can stream media using RTP and RTSP over HTTP. This means users can view media through most firewalls.
    • Relay support. You can replicate content for live broadcasts. With relays, you can scale streaming services to reach more viewers.
    • Enhanced logging. Darwin Streaming Server software now supports third-party analysis tools that provide consolidated server reporting.
    • Authentication support. You can control access to media files using the authentication modules that come with Darwin Streaming Server software, or you can supply your own module.
    • Module API. The Darwin Streaming Server software now supports dynamically loadable modules which can extend the functionality of the server through the QTSS API.
