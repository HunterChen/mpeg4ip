Setting up a Web page to show streamed media

    You can embed streamed media in a Web page. If you do this, viewers can use any Web browser to view the media by entering the URL of the Web site.

    For example:

    http://www.mywebpage.com/

    You'd replace "www.mywebpage.com" with the DNS name of your Web site.

    To embed live media in a Web page, you must create a reference movie that points to the SDP file in the directory you set up for streaming media. You store the reference movie on your Web site and include a link to the reference movie on your Web page.

    For information on creating reference movies, see the following site:
    • QuickTime authoring Web site
    www.apple.com/quicktime/authoring
