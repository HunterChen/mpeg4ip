Time-to-live value

    A time-to-live value is used with multicasts to specify the number of times a media stream can be passed from one router to another before the stream is no longer transmitted.

    The value can be any number between 0 and 255. A value of 1 reaches client computers on the local area network. The larger the number, the farther the multicast packets will travel.
