Working with playlist settings

To see playlist settings, click Settings in Streaming Server Admin, click Playlists Settings, and then click the name of the playlist whose settings you want to see.

"Playlist Name" is the name you give your playlist and the name that appears in the Playlists Settings pane.

"Playlist URL" is the name you give to clients who want to view the playlist broadcast (or that you embed in a Web page). It is the same as the name in the Playlist Name field. Spaces are automatically converted to underscores and ".sdp" is appended to the name of the file. You don't have to enter a playlist URL. It is generated when you click Submit. If you want to change it later, you can.

"Play Mode" can be one of three choices:
• "Sequential" broadcasts the media in the order it appears in the playlist file. When the last media file is done playing, the broadcast stops.
• "Sequential Looped" broadcasts the media in the order it appears in the playlist file. When the last media file is done playing, the playlist repeats in the same order.
• "Weighted Random" broadcasts the media in random order using the specified weights to determine how often an item plays. The higher the number, the more weight it carries. The media continues to play in random order until you stop the broadcast.

"Items only repeat after _ other items have played" lets you set the number of items that must play before an item can repeat. If you set a value other than zero for repeated items, make sure the number is less than the number of media files in the playlist.

Click the Add/Remove Items button to add items to your playlist. After you're done, make sure you return to the Details pane to set the order and weight for each item.

You can also enable logging for each playlist by clicking the Enable Logging checkbox.
