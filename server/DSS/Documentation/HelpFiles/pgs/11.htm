Working with Logging settings

To see the Logging settings, click Settings in Streaming Server Admin, then click Logging Settings.

You can adjust these settings by changing information in the fields or clicking the preferred button. You can specify that the log be reset after a certain number of days or after it reaches a certain size (in Kb).

"Error Log" shows error messages. Use this log to troubleshoot problems with the server.

"Access Log" shows the number of times each media file has been accessed, when it was accessed, and who accessed it since the log was reset. Access errors are also reported in the log.
