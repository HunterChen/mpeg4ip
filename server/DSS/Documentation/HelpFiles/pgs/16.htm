Creating or changing a playlist

To create or edit a playlist:

1 Click Settings in Streaming Server Admin, then click Playlists Settings.
2 Click Create New Playlist, or click a playlist name.
3 Enter a name, if this is a new playlist.
4 Use the pop-up menu to set a play mode.
5 Click Add/Remove Items to add or remove items from the playlist.
6 Use the Details pane to establish the order and weight for items (if broadcast randomly), and to enable logging for the playlist.
7 Click Submit to save your changes and save the playlist.
