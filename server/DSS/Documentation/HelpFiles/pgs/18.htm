Adding items to a playlist

You add items to a playlist by clicking the Add/Remove Items button in the Details pane for the playlist.

To add items to a playlist:

1 Click Settings in Streaming Server Admin, then click Playlists Settings.
2 Click Create New Playlist, or click a playlist name.
3 Click Add/Remove Items.
4 Select a directory that contains the items you want to broadcast.
5 Click the box in the Add column next to each item you want to appear in the playlist.
6 Click Add to Playlist.
