Viewing access and error logs

    If logging is turned on in the Settings panel in Streaming Server Admin, information about access and errors is saved to log files.

    To open a log:
    1 Click Logs.
    2 Click Error Log or Access History to choose which log you want to view.

Analyzing the access log

    The information that's stored in the access log can be analyzed using third-party software.
