Viewing streamed media from a client computer

Users can view media streamed from your server using software that supports QuickTime 4, such as QuickTime Player. To view streaming media, users must have QuickTime 4 or later installed.

To view media from a client computer:

1 Open QuickTime Player.
2 Choose Open URL from the File menu.
3 Enter the URL for the media file (for example: rtsp://myserver.com/mymedia).

Replace "myserver.com" with the DNS name of your streaming server computer, and "mymedia" with the name of your hinted movie or media file.

If you want users to view streamed media through a Web browser, you must set up a Web page to show the media. Then give users the URL of the Web page.
