Configuring streaming services

    To configure streaming services:
    1 Open the /etc/streamingserver.conf file.
    You may need to enter the Administrator (root) password to make changes to this file.
    2 Locate the item(s) you want to change.
    3 Locate the movie_folder parameter and examine the path to the directory where the streamed media will be stored (or the Session Description Protocol [SDP] files, if you are streaming live audio or video).
    You can use the default location (/usr/movies) by leaving the parameter value unchanged, or you can choose another location.
    4 Select other options you want to use.
    5 Save your changes to the configuration file.
    6 Start the server.
