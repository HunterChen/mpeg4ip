If users see error messages while streaming media

Here's what the different messages mean:
• Error code 401: the user has tried to get access to a protected file. Users may need to upgrade to QuickTime 5.0 or later.
• Error code 404: the server cannot locate the URL the user entered. Make sure users enter the correct URL for the broadcast.
• Error code 415 indicates the movie file is not hinted or has a compressed movie resource. You need to hint the movie again using the Pro version of QuickTime Player.
• Error code 453: the server is too busy for users to view the stream. Users should try again later. You may want to increase the maximum number of connections in the General Settings pane of Streaming Server Admin.
• Error code 454: the connection to the server was dropped. Users must start viewing the stream again.
• Error code -5420: may indicate that the server is no longer running. Check whether the streaming server is running, and restart it if necessary.
